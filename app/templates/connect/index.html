Voor wie bouwt

Verbind je AI-tool.

Koppel dewereldvan.ai aan Claude Code, Cursor of je eigen agent — en praat er rechtstreeks mee. Bouw je profiel uit een link, doorzoek de makers, haal je matches op en stel je voor, zonder je editor te verlaten.

1
Genereer een token

Eén token per tool. Je ziet 'm één keer — bewaar 'm goed.

2
Voeg de server toe aan je tool

In Claude Code, met je token uit stap 1:

claude mcp add --transport http dewereldvan \
  {{ mcp_url }} \
  --header "Authorization: Bearer <jouw-token>"

Daarna kun je gewoon vragen: "voeg mijn project X toe aan dewereldvan" of "laat mijn matches zien".
